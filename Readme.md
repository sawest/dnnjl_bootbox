<p>Tests are run using <a href="http://karma-runner.github.io/0.8/index.html">Karma</a> using the Mocha test adapter.
To run the tests yourself, simply run <code>npm install</code> within the project followed by <code>npm test</code>.
Please note that this will require <a href="http://phantomjs.org/">PhantomJS</a> being installed and in your path - if
it is not, you may run the tests and capture browsers manually by running <code>karma start</code> from the root
of the project.</p>

<p>The project is also hosted on <a href="https://travis-ci.org/makeusabrew/bootbox">Travis CI</a> - when submitting
pull requests <strong>please</strong> ensure your tests pass as failing requests will be rejected. See the
<a href="https://github.com/makeusabrew/bootbox/blob/master/CONTRIBUTING.md">CONTRIBUTING</a> file for more information.</p>

<h2><a id="user-content-building-a-minified-release" class="anchor" href="#building-a-minified-release" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Building a minified release</h2>

<p>The repository no longer contains a minified bootbox.min.js file - this is now only generated
<a href="https://github.com/makeusabrew/bootbox/releases">for releases</a>. To build your own minified copy
for use in development simply run <code>npm install</code> if you haven't already, followed by <code>grunt uglify</code>.
This will generate a bootbox.min.js file in your working directory.</p>

<h2><a id="user-content-a-note-on-bootstrap-dependencies" class="anchor" href="#a-note-on-bootstrap-dependencies" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>A note on Bootstrap dependencies</h2>

<p>Bootbox <strong>4.0.0</strong> is the first release to support Bootstrap 3.0.0.</p>

<p>Bootbox <strong>3.3.0</strong> is the <em>last</em> release to support Bootstrap 2.2.x.</p>

<p>Much more dependency information can be found <a href="http://bootboxjs.com/#dependencies">on the Bootbox website</a>.</p>

<h3><a id="user-content-roadmap" class="anchor" href="#roadmap" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Roadmap</h3>

<p>The latest major release of Bootbox - 4.0.0 - involved a total rewrite of the
internal code and introduced an entirely new public API. It has not re-implemented
some functionality from the 3.x series as of yet; this will be addressed in the
form of new minor releases. Please feel free to add feedback and requests.</p>

<p>There is no new major (e.g. 5.x) release on the roadmap at present.</p>

<h3><a id="user-content-latest-release-440" class="anchor" href="#latest-release-440" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Latest Release: 4.4.0</h3>

<ul>
<li>Allow <code>backdrop</code> options of <code>true</code> and <code>false</code> to dismiss modals</li>
<li>Pass dialog as <code>this</code> value in callbacks</li>
<li>Bootstrap 3.3.2 compatibility</li>
<li>jQuery 1.11.2 compatibility</li>
<li>Add support for <code>maxlength</code> prompt input attribute</li>
<li>Gracefully detect lack of Bootstrap library rather than crashing</li>
<li>Expose <code>addLocale</code> and <code>removeLocale</code> for custom locale settings</li>
<li>Expose <code>setLocale</code> helper to select a locale rather than using <code>setDefaults("locale", ...)</code></li>
<li>Add Hungarian locale</li>
<li>Add Croatian locale</li>
<li>Add Bulgarian locale</li>
<li>Add Thai locale</li>
<li>Add Persian locale</li>
<li>Add Albanian locale</li>
</ul>

<p>For a full list of releases and changes please see <a href="https://github.com/makeusabrew/bootbox/blob/master/CHANGELOG.md">the changelog</a>.</p>
